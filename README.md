# PDF Grid Snapshotter

A Python tool and library to generate beautiful, slide-ready grid snapshots ("grids eye view") of PDF documents. It automatically optimizes the grid dimensions to match a target aspect ratio (such as the landscape golden ratio) and adds visual enhancements like drop shadows, custom background colors, and page/grid demarcation.

## Features

- **Optimal Grid Layout Calculation**: Automatically calculates the best row and column configurations to fit all pages while minimizing deviation from a target aspect ratio (default: `1.618` - landscape golden ratio).
- **High-Resolution Page Downsampling**: Renders pages at a high quality DPI (e.g. 150 DPI) and downsamples to target thumbnail widths using Lanczos resampling, resulting in readable and crisp text/diagrams without pixelated aliasing.
- **Visual Enhancements**:
  - Soft 3D drop shadows under each page for depth.
  - Thin borders around pages to make them stand out.
  - Configurable canvas background colors.
- **Page Demarcation (Highlights)**:
  - **Page ranges**: Outline individual pages by index (e.g., `-p 1-3,5`).
  - **Grid blocks**: Highlight rectangular blocks of pages in grid coordinates (e.g., `-g 1,1,7,2` to outline a bounding box enclosing rows 1 to 7 and columns 1 to 2).
  - **Custom styles**: Change outline color, width, and padding (e.g., `-s "color=blue,width=4,padding=6"`).
- **Preset Spacings**:
  - `reference`: Uses the layout gap proportions of the reference image (larger horizontal spacing, compact vertical spacing).
  - `compact`: A tight, uniform spacing grid.
  - `custom`: Manually specify horizontal/vertical gaps and margins.

## Command Line Usage

Use the `pdf_snapshot_tool` executable or run the module directly.

```bash
./pdf_snapshot_tool <input_pdf_path> <output_png_path> [options]
```

### Examples

#### 1. Recreate the reference image exactly
To generate a snapshot of the `2601.20927v1.pdf` matching the `reference.png` layout (7x10 grid with columns 1 and 2 outlined in red):
```bash
./pdf_snapshot_tool 2601.20927v1.pdf output_grid.png \
  --page-width 96 \
  --preset reference \
  --demarcate-grid-block 1,1,7,2
```

#### 2. Highlight specific page numbers with a custom blue border
Highlight pages 1, 2, 3, and 5 with a blue border of width 4 and padding 6:
```bash
./pdf_snapshot_tool 2601.20927v1.pdf output_grid.png \
  --page-width 120 \
  --demarcate-pages "1-3,5" \
  --demarcate-style "color=blue,width=4,padding=6"
```

#### 3. Customize layout aspect ratio and background color
Create a layout matching standard 16:9 widescreen ratio with a white background and compact spacing:
```bash
./pdf_snapshot_tool 2601.20927v1.pdf output_grid.png \
  --ratio 1.777 \
  --preset compact \
  --bg-color white
```

#### 4. Custom pixel spacing and margin override
```bash
./pdf_snapshot_tool 2601.20927v1.pdf output_grid.png \
  --preset custom \
  --gap-x 30 \
  --gap-y 30 \
  --margin 50
```

## Python API Usage

You can also use this library directly inside Python scripts:

```python
from pdf_snapshot import create_snapshot

rows, cols, final_ratio, img_w, img_h = create_snapshot(
    pdf_path="2601.20927v1.pdf",
    output_path="output_grid.png",
    target_ratio=1.618,
    page_width=120,
    render_dpi=150,
    bg_color="#f0f2f5",
    layout_preset="reference",
    demarcated_pages=[1, 2, 3, 5],
    demarcation_style={"color": "blue", "width": 4, "padding": 6}
)

print(f"Generated a {rows}x{cols} grid snapshot ({img_w}x{img_h}) with ratio {final_ratio:.3f}.")
```

## API Reference

### `create_snapshot(...)`
Exposes the core grid layout and snapshotting rendering:

| Parameter | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `pdf_path` | `str` | *Required* | Path to the input PDF file. |
| `output_path` | `str` | *Required* | Path where the output PNG should be saved. |
| `target_ratio` | `float` | `1.618` | Target aspect ratio of the grid (landscape golden ratio). |
| `page_width` | `int` | `120` | Width of each page thumbnail in pixels. Height scales automatically. |
| `render_dpi` | `int` | `150` | Resolution for rendering PDF pages before scaling down. |
| `bg_color` | `str`/`tuple` | `"#f0f2f5"` | Canvas background color. |
| `layout_preset` | `str` | `"reference"` | Preset spacing: `"reference"`, `"compact"`, or `"custom"`. |
| `custom_gap_x` | `int`/`None` | `None` | Horizontal spacing between page columns (for `"custom"` preset). |
| `custom_gap_y` | `int`/`None` | `None` | Vertical spacing between page rows (for `"custom"` preset). |
| `custom_margin` | `int`/`None` | `None` | Margin around the canvas boundary (for `"custom"` preset). |
| `demarcated_pages` | `list` | `[]` | List of 1-indexed page indices to highlight. |
| `demarcated_grid_blocks` | `list` | `[]` | List of tuples `(r1, c1, r2, c2)` specifying grid blocks to highlight. |
| `demarcation_style` | `dict` | `{}` | Style configurations: `{"color": "#dc3545", "width": 3, "padding": 4}`. |
